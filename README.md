# Campos-Flavia-Ejercicio-Final

Este proyecto tiene como objetivo analizar un corpus de textos en español. Se realiza limpieza de palabras, eliminación de stopwords, conteo de frecuencias, visualizaciones y generación de resultados por archivo de texto.   
Herramientas utilizadas: Voyant Tools y R.

## Estructura del proyecto

- `corpus2` Carpeta que contiene los archivos de texto (*.txt).  
- `Voyant Tools`  Imágenes, visualizaciones.
- `Campos, Flavia -Análisis del Corpus.docx` Análisis de resultados.
- `Campos-EjercicioFinal.R` Código de R.
- `README.md`  Este archivo, con la descripción y guía del proyecto.
